Projects

Traffic Hub

Traffic Hub is a demonstrative web app that presents real-time road camera images and car parks vacancy data. Google Map API and data APIs from the Hong Kong Government are intergrated, to create a controllable map that lets users select specific spot. When viewing with certain screen sizes, the images/texts are shown inside the map as a pop up bubble instead.

  • JavaScript
  • Google Map API
  • JSON
  • CSS

EaZzy Upload

EaZzy Upload is a demonstrative storage website. With an user-friendly drag and drop interface, users can upload files and a link for later download is generated. Photos are stored in localhost while related informations are stored in a MySQL database.

  • JavaScript
  • MySQL
  • PHP
  • CSS

Hidden Secrets

Hidden Secrets is a computer game project that was nominated for best final year project award and rated A- grade. Although not directly related to web development, I believe it shows my creativity and programming skills were improved and put into practice.
